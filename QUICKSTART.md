# 快速啟動指南

## 第一次使用

### 1. 安裝後端依賴

```bash
cd backend
npm install
```

### 2. 安裝前端依賴

```bash
cd ../frontend
npm install
```

## 啟動系統

### 方法一：分別啟動（推薦）

**終端機 1 - 啟動後端**
```bash
cd backend
npm run dev
```

後端會在 `http://localhost:3001` 啟動

**終端機 2 - 啟動前端**
```bash
cd frontend
npm run dev
```

前端會在 `http://localhost:5173` 啟動

### 方法二：使用提供的啟動腳本

在專案根目錄執行：

```bash
# macOS/Linux
chmod +x start.sh
./start.sh
```

## 使用系統

1. 開啟瀏覽器訪問 `http://localhost:5173`
2. 點擊「選擇檔案」上傳 Excel 檔案
3. 點擊「上傳並處理」
4. 等待處理完成
5. 點擊「下載整理後的 Excel」

## 常見問題

### Q: 前端無法連接後端？
A: 確認後端服務已啟動在 `http://localhost:3001`，可以訪問 `http://localhost:3001/health` 檢查

### Q: 上傳失敗？
A: 
- 確認檔案格式是 .xlsx 或 .xls
- 確認檔案包含所有必要欄位
- 檢查檔案大小是否超過 10MB

### Q: 處理後的檔案在哪裡？
A: 點擊「下載整理後的 Excel」按鈕，檔案會自動下載到您的下載資料夾

## 停止服務

在兩個終端機中按 `Ctrl + C` 即可停止服務
