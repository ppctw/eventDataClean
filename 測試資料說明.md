# 測試資料說明

## 建立測試 Excel 檔案

由於程式無法直接建立 Excel 檔案，請您手動建立一個測試檔案：

### 方法一：使用 Excel 或 Numbers

1. 開啟 Excel 或 Numbers
2. 建立新的試算表
3. 在第一列輸入以下欄位名稱（必須完全一致）：

```
報名序號 | 兒童姓名 | 性別 | 年級 | 學校 | 家長姓名 | 家長行動電話 | 備註
```

4. 從第二列開始輸入測試資料，例如：

| 報名序號 | 兒童姓名 | 性別 | 年級 | 學校 | 家長姓名 | 家長行動電話 | 備註 |
|---------|---------|-----|------|------|---------|-------------|------|
| 001 | 王小明 | 男 | 三年級 | XX國小 | 王大華 | 0912345678 | |
| 002 | 王小美 | 女 | 一年級 | XX國小 | 王大華 | 0912345678 | |
| 003 | 李小華 | 男 | 二年級 | YY國小 | 李大明 | 0923456789 | |
| 004 | 張小芳 | 女 | 三年級 | ZZ國小 | 張大偉 | 0934567890 | |
| 005 | 張小強 | 男 | 一年級 | ZZ國小 | 張大偉 | 0934567890 | |
| 006 | 陳小玲 | 女 | 二年級 | AA國小 | 陳大成 | 0945678901 | |
| 007 | 林小傑 | 男 | 三年級 | BB國小 | 林大雄 | 0956789012 | |
| 008 | 黃小婷 | 女 | 一年級 | CC國小 | 黃大明 | 0967890123 | |

5. 儲存為 `.xlsx` 格式

### 方法二：使用 Google Sheets

1. 開啟 Google Sheets
2. 建立新的試算表
3. 輸入上述資料
4. 下載為 Excel 格式（檔案 → 下載 → Microsoft Excel (.xlsx)）

## 測試重點

### 1. 手足關係測試

在上述範例中：
- **王小明** 和 **王小美** 是手足（家長：王大華）
- **張小芳** 和 **張小強** 是手足（家長：張大偉）
- 其他孩子沒有手足

### 2. 年級分組測試

系統會建立以下分頁：
- **一年級**：王小美、張小強、黃小婷
- **二年級**：李小華、陳小玲
- **三年級**：王小明、張小芳、林小傑

## 測試步驟

1. 建立上述測試 Excel 檔案
2. 開啟瀏覽器訪問 http://localhost:5173
3. 點擊「選擇檔案」上傳測試檔案
4. 點擊「上傳並處理」
5. 等待處理完成
6. 點擊「下載整理後的 Excel」
7. 開啟下載的檔案，檢查：
   - 是否有三個分頁（一年級、二年級、三年級）
   - 每個分頁的項次是否從 1 開始
   - 手足資訊是否正確
   - 資料是否完整

## 進階測試

### 測試空值處理

建立包含以下情況的測試資料：
- 年級為空（應歸類到「未分類」分頁）
- 家長姓名為空（手足欄位應顯示「無」）
- 備註為空（應正常處理）

### 測試多個手足

建立一個家長有 3 個以上孩子的資料，檢查手足資訊是否用逗號正確分隔。

例如：
```
009 | 劉小一 | 男 | 一年級 | DD國小 | 劉大富 | 0978901234 |
010 | 劉小二 | 女 | 二年級 | DD國小 | 劉大富 | 0978901234 |
011 | 劉小三 | 男 | 三年級 | DD國小 | 劉大富 | 0978901234 |
```

劉小二的手足欄位應顯示：
- 手足名稱：劉小一, 劉小三
- 手足性別：男, 男
- 手足年級：一年級, 三年級

## 常見問題排除

### Q: 上傳失敗？
- 確認檔案格式是 .xlsx 或 .xls
- 確認檔案大小不超過 10MB
- 確認欄位名稱完全一致（包括空格）

### Q: 手足資訊不正確？
- 檢查「家長姓名」欄位是否有拼字錯誤
- 確認家長姓名沒有多餘的空格

### Q: 某些資料沒有出現？
- 檢查「年級」欄位，為空的資料會在「未分類」分頁
