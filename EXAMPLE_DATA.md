# Excel 範例資料格式

## 輸入檔案格式

您的 Excel 檔案應該包含以下欄位（第一列為標題列）：

### 必要欄位

| 欄位名稱 | 說明 | 範例 |
|---------|------|------|
| 報名序號 | 報名的流水號 | 001, 002, 003 |
| 兒童姓名 | 參加活動的兒童姓名 | 王小明 |
| 性別 | 兒童性別 | 男、女 |
| 年級 | 兒童就讀年級 | 一年級、二年級、三年級 |
| 學校 | 就讀學校名稱 | XX國小 |
| 家長姓名 | 家長或監護人姓名 | 王大華 |
| 家長行動電話 | 聯絡電話 | 0912345678 |
| 備註 | 其他備註資訊 | （可空白） |

## 範例資料

```
報名序號  兒童姓名  性別  年級    學校      家長姓名  家長行動電話  備註
001      王小明    男   三年級   XX國小    王大華   0912345678
002      王小美    女   一年級   XX國小    王大華   0912345678
003      李小華    男   二年級   YY國小    李大明   0923456789
004      張小芳    女   三年級   ZZ國小    張大偉   0934567890
005      張小強    男   一年級   ZZ國小    張大偉   0934567890
```

## 手足判斷邏輯

系統會依據「家長姓名」來判斷手足關係：

- **王小明** 和 **王小美** 的家長都是「王大華」→ 他們是手足
- **張小芳** 和 **張小強** 的家長都是「張大偉」→ 他們是手足
- **李小華** 的家長是「李大明」→ 沒有手足

## 輸出檔案格式

處理後的 Excel 會依「年級」分成多個分頁：

### 分頁範例

**一年級** 分頁：
```
項次  報名序號  兒童姓名  性別  年級    學校    手足名稱  手足性別  手足年級  家長姓名  家長行動電話  備註
1    002      王小美    女   一年級   XX國小   王小明    男       三年級    王大華   0912345678
2    005      張小強    男   一年級   ZZ國小   張小芳    女       三年級    張大偉   0934567890
```

**二年級** 分頁：
```
項次  報名序號  兒童姓名  性別  年級    學校    手足名稱  手足性別  手足年級  家長姓名  家長行動電話  備註
1    003      李小華    男   二年級   YY國小   無        無       無        李大明   0923456789
```

**三年級** 分頁：
```
項次  報名序號  兒童姓名  性別  年級    學校    手足名稱  手足性別  手足年級  家長姓名  家長行動電話  備註
1    001      王小明    男   三年級   XX國小   王小美    女       一年級    王大華   0912345678
2    004      張小芳    女   三年級   ZZ國小   張小強    男       一年級    張大偉   0934567890
```

## 注意事項

1. **欄位名稱必須完全一致**：系統會依照欄位名稱來讀取資料
2. **年級為空**：如果年級欄位為空，該筆資料會被歸類到「未分類」分頁
3. **家長姓名為空**：無法判斷手足關係，手足欄位會顯示「無」
4. **多個手足**：如果有多個手足，會用逗號分隔顯示
5. **檔案格式**：支援 .xlsx 和 .xls 格式
6. **檔案大小**：限制 10MB 以內

## 建立測試資料

您可以使用 Excel 或 Google Sheets 建立測試資料：

1. 建立新的試算表
2. 在第一列輸入欄位名稱（如上表）
3. 從第二列開始輸入資料
4. 儲存為 .xlsx 格式
5. 上傳到系統進行測試
